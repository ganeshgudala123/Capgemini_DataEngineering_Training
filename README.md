# Data Engineering Training Repository

This repository contains all the concepts and practical implementations covered during the Data Engineering Training program.

## 📅 WEEK 0

### 🔹 Phase 0
- Completed the Databricks Associate Data Engineering course.

### 🔹 Phase 1

📚 **Topics Covered:**
- SELECT
- FILTER
- GROUP BY

✅ **Key Learnings:**
- Understanding GROUP BY and its applications  
- Mapping SQL queries to PySpark DataFrame operations  

⚠️ **Challenges Faced:**
- Understanding GROUP BY concepts  
- Writing queries in PySpark

### 🔹 Phase 2

📚 **Topics Covered:**
- Joins and aggregations in PySpark  
- Mapping SQL queries to DataFrame operations  

✅ **Key Learnings:**
- Learned to use `join()`, `groupBy()`, `sum()`, `avg()`, `filter()`, and `orderBy()`  
- Understood how SQL logic translates to PySpark  

⚠️ **Challenges Faced:**
- Writing correct join conditions  
- Understanding different join types  
- Applying multiple aggregations

### 🔹 Phase 3

📚 **Topics Covered:**
- ETL pipeline (Extract → Transform → Load)  
- Data ingestion from CSV files  
- Data cleaning (null handling)  
- Schema validation  
- Joins and aggregations  
- Pipeline-based transformations in PySpark  

✅ **Key Learnings:**
- Reading and processing real-world datasets  
- Handling missing values using `dropna()` and `fillna()`  
- Converting data types using `cast()`  
- Building structured pipelines using `join()`, `groupBy()`, `sum()`, `count()`  
- Understanding end-to-end PySpark workflows  

⚠️ **Challenges Faced:**
- Handling null values  
- Converting incorrect data types  
- Writing proper join conditions  
- Understanding ETL pipeline flow  

---

### 🔹 Phase 4

📚 **Topics Covered:**
- End-to-end data pipeline creation  
- Data cleaning (nulls, duplicates, invalid values)  
- Joins and aggregations  
- Customer segmentation  
- Final reporting using PySpark  

✅ **Key Learnings:**
- Building complete business pipelines  
- Performing data cleaning before transformations  
- Calculating metrics (daily sales, city-wise revenue, customer spend)  
- Implementing segmentation (Gold, Silver, Bronze)  
- Combining transformations into final reporting datasets  

⚠️ **Challenges Faced:**
- Handling null keys before joins  
- Choosing correct join order  
- Managing duplicates and invalid data  
- Writing aggregation logic  
- Combining multiple outputs into one dataset
